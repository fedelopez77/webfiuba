Projects


  • Summa NLP - Summarizer

    • Summa is a text summarizer developed as a Software Engeneering final project. It uses an improvement of the TextRank algorithm introduced by Rada Mihalcea and Paul Tarau on their paper: TextRank: Bringing Order Into Texts. Summa is open source software released under the MIT License. Its entire source code is hosted on github.
